## 三種 AI 的真正分別

```
🔍 搜索型 AI（Perplexity、Grok）
   → 就係「超強 Google」
   → 你搜嘢，它幫你整理答案
   → 你唔問 = 它睡著，完全唔主動

💬 對話型 AI（ChatGPT、Claude、Gemini）
   → 就係「超聰明打字機」
   → 你問問題，它回答、寫文章、分析資料
   → 但係你唔開 App、唔問它 = 它唔存在
   → 每次關掉重開，它唔記得你係誰

🤖 代理型 AI（Area2 AI）← 我哋係呢類
   → 就係「住喺你 WhatsApp 的全職員工」
   → 24 小時在線，你唔在它都在做嘢
   → 記住你的業務、客戶、習慣
   → 主動提醒你、自動執行、越用越識你
```

### 差異 2：「越用越識你」——記憶系統

**競品的問題：**
每次打開 ChatGPT，它唔記得你係誰。你要重新解釋：「我係做貿易嘅，主要客係中東市場，我公司叫...」
用了兩年 ChatGPT，它仍然不認識你。

**Area2 AI 的做法（向量記憶庫）：**

普通人解釋：「向量記憶庫」就係AI 的長期記憶，功能就像一個非常聰明的秘書——她唔只記住你說過的事，更懂得理解背後的意思，在相關情況主動提醒你。

**記憶是怎樣運作的：**
```
第 1 次：
你：「我叫陳生，公司做貿易，主要出口中東」
AI：記住 ✅

第 10 次：
你：「幫我寫封回覆客戶的電郵」
AI：自動用陳生的語氣、公司背景、中東市場文化寫 ✅
（你唔使再解釋係誰）

第 30 次：
你：「今日收到個新客戶查詢」
AI：「這個客戶是中東市場嗎？要我用您慣用的阿拉伯語問候語開頭嗎？」
（AI 主動聯想，唔使你再指示）✅

第 90 次：
AI 已記住：
- 你的 50 個重要客戶名字和習慣
- 你常用的回覆格式和語氣
- 你的業務淡旺季和重要 Deadline
- 你偏好的工作方式

**為什麼越用越有價值（留客護城河）：**

**競品的問題：**
ChatGPT、Perplexity、Gemini 全部都係「你問才有」——你不在線，它就停擺。沒有一個可以「定時主動做嘢」。

```

→ 競品在小紅書發新帖 → 即時通知你
→ 客戶合約 30 日後到期 → 提前提醒你
```

**真實場景：**

### 差異 4：「多個 AI 腦同時工作」——Sub-Agent 系統

**用普通人的話解釋（唔需要知道「Multi-Agent」係咩）：**

想像你請了一個超級能幹的項目經理，她接到你的大任務後，不是自己一個人慢慢做，而是**立即分配給幾個專門的助手同時進行**，然後整合所有結果交給你。

**競品的問題：**
ChatGPT 是「獨行俠」——一次只能做一件事，任務越複雜越慢，甚至做唔到。

**Area2 AI 的做法：**
```
你（WhatsApp）：「幫我做一份競爭對手報告」

Area2 AI（主腦）接收任務後：
├── 助手A：同時搜索競品官網 ──────────→ 結果
├── 助手B：同時爬取競品社交媒體 ────→ 結果  } 全部
├── 助手C：同時分析競品定價策略 ────→ 結果  } 同時
└── 助手D：同時整理行業最新新聞 ────→ 結果    } 進行

主腦整合所有結果 → 生成報告 → WhatsApp 給你
（完成時間：15分鐘 vs 獨行俠需要1小時）
```

**實際應用：**
- 做市場研究報告（正是這份計劃書的製作方式）
- 同時測試多個網頁
- 批量處理大量文件
- 同時監察多個社交媒體平台

### 差異 6：「MCP 工具」——AI 自己給自己升級

**用普通人的話解釋（「MCP」係咩）：**

MCP（Model Context Protocol）係一個國際標準，作用就係讓 AI 可以連接任何外部系統。

打個比喻：  
- **Skills 技能包** = 員工已學會的技能（懂得做的事）  
- **MCP 工具** = 員工的工具箱（用來做事的工具）

具體例子：
```
Skills（員工技能）：
→「我識幫你查 HKTVMall 的價格」（知道怎麼做）

MCP 工具（工具）：
→ 連接到 HKTVMall 系統的「查詢器」（做事的工具）
→ 連接到你公司 ERP 的「讀取器」
→ 連接到 Google Calendar 的「行事曆器」
```

**Skills vs MCP — 甚麼時候用哪個：**

| 情況 | 用 Skills | 用 MCP |
|------|----------|-------|
| 想要一個可重複使用的工作流程 | ✅ | |
| 想連接一個外部系統（如 ERP、CRM）| | ✅ |
| 想讓 AI 讀取你公司的 Google Sheets | | ✅ |
| 想自動化一個複雜的業務流程 | ✅ | |
| 想讓 AI 控制瀏覽器做網頁操作 | | ✅ |
| 唔知用邊個？ | **直接問 Area2 AI，它會建議** | |

**Area2 AI 獨特之處——AI 可以幫自己安裝新工具：**

```
你：「我想讓你連接我們公司的 Xero 會計系統」
AI：「好的，我幫你安裝 Xero MCP 工具，
     安裝後我就可以直接讀取你的發票和帳目」
    → AI 自動設定連接 → 下次你問帳目問題，它直接從 Xero 抓數據

**已支援的 MCP 工具例子：**
